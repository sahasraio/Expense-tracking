Project Overview

The Expense Tracking and Budget Management System is a web-based application designed to help users manage their personal finances efficiently. The system allows users to record daily income and expenses, categorize transactions, and monitor their financial balance in real time. The application provides a simple and user-friendly interface for maintaining financial records digitally instead of using manual methods.

This project demonstrates the practical implementation of Front-End Web Development concepts and Data Structures by using HTML, CSS, and JavaScript to store, process, and display financial data.

Features

User Login Authentication

Add income and expense transactions

Categorize expenses (Food, Travel, Bills, Shopping, Entertainment)

Real-time balance calculation

Transaction deletion functionality

Dark mode interface

Data persistence using Local Storage

Responsive and user-friendly dashboard

Technologies Used

HTML5 – Structure of the web application

CSS3 – Styling and responsive design

JavaScript – Application logic and functionality

Local Storage – Client-side data storage

System Modules

User Authentication Module
Provides a basic login system to restrict access to authorized users.

Expense Management Module
Allows users to add and delete transactions including income and expenses.

Budget Tracking Module
Calculates and displays total income, total expenses, and current balance.

Expense Categorization Module
Classifies expenses into categories for better financial understanding.

Data Storage Module
Stores all financial records using browser local storage for persistence.

How to Run the Project

Download the project folder.

Open the file index.html in any web browser.
